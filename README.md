Reevio
======

Installation
------------

1. Download [Composer](http://getcomposer.org)

2. Move into the folder you'd like to install Reevio in 

       cd your/reevio/path
			
3. Install Composer

       php composer.phar install
			
4. Rename app.ini.sample to app.ini
5. Fill in your information in app.ini
